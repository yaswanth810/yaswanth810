### 🔐 [EduTrust — Blockchain Certificate Platform](https://github.com/yaswanth810/certificate-validation-for-academic)
> Production-ready on-chain academic credential system on Ethereum Sepolia

- **3 smart contracts**: `CertificateNFT.sol` (ERC-721), `ScholarshipEscrow.sol` (time-locked funds), `VignanRegistry.sol` (student registry)
- RBAC with `ADMIN_ROLE`, `MINTER_ROLE`, `SCHOLARSHIP_MANAGER_ROLE` via OpenZeppelin v5
- IPFS storage via Pinata with CID-based verification + QR code validation
- OCR-powered legacy certificate digitization (AI-assisted form auto-fill)
- Multi-wallet support via Web3Modal v3 · real-time event listeners · auto network switching
